**ARISE Platformasi** — talabalarning ilmiy-tadqiqotchilik salohiyatini rivojlantirish uchun yaratilgan maxsus web-ekotizim. Ushbu platforma ilmiy materiallar, interaktiv testlar va natijalarni monitoring qilish tizimini o'z ichiga oladi.

---

## 🌟 Asosiy Imkoniyatlar

### 📚 O'quv Modullari (Hierarchical Structure)
Loyiha **Modul -> Mavzu -> Blok** tizimida tuzilgan bo'lib, har bir ilmiy mavzu bo'yicha batafsil materiallar, Word hujjatlar va metodik qo'llanmalar taqdim etiladi.

### 📝 Interaktiv Testlar
Turli ilmiy yo'nalishlar (Academic Freedom, Research Ethics va h.k.) bo'yicha bilimlarni sinash uchun mo'ljallangan test tizimi.

### 📊 Admin Dashboard & Natijalar
Faqat administratorlar uchun mo'ljallangan maxsus natijalar paneli orqali barcha foydalanuvchilarning natijalarini kuzatish va tahlil qilish imkoniyati.

---

## 🛠 Texnologik Stek

* **Backend:** Python 3.13, Django Framework
* **Frontend:** Bootstrap 5.3, Bootstrap Icons, CSS3 (Custom animations)
* **Database:** SQLite (Yengil va samarali)
* **Security:** CSRF protection, Secure Logout (POST method), Admin access control
