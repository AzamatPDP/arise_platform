**ARISE Platformasi** — talabalarning ilmiy-tadqiqotchilik salohiyatini rivojlantirish uchun yaratilgan zamonaviy web-ekotizim. Loyiha ilmiy materiallarni tizimlashtirish, interaktiv testlar orqali bilimlarni baholash va natijalarni real vaqt rejimida monitoring qilish imkonini beradi.

---

## 🌟 Asosiy Imkoniyatlar

### 🎨 Clean & Responsive UI
Platforma foydalanuvchi tajribasiga (UX) asoslangan moviy-oq interfeysga ega. Bosh sahifa orqali barcha asosiy bo'limlarga (Testlar, Modullar, Taqdimotlar) bir marta bosish orqali o'tish mumkin.

### 📊 Professional Natijalar Paneli
Administratorlar uchun yaratilgan maxsus interfeys orqali test natijalarini kuzatish, ballarni tahlil qilish va foydalanuvchilar faolligini nazorat qilish imkoniyati mavjud.

### 📂 Ierarxik Ma'lumotlar Tuzilmasi
Ilmiy kontent **Modul ➡️ Mavzu ➡️ Blok** ierarxiyasi asosida tashkil etilgan bo'lib, bu ma'lumotlarni oson o'zlashtirishga xizmat qiladi.

## 🛠 Texnologik Stek

* **Backend:** Python 3.13, Django 6.0 Framework
* **Frontend:** HTML5, CSS3 (Custom animations), Bootstrap 5.3
* **Security:** * Xavfsiz Logout (POST method)
    * CSRF Protection & Admin Access Control
    * `@user_passes_test` dekoratorlari orqali himoyalangan natijalar paneli
* **Database:** SQLite (Lightweight & Portable)
